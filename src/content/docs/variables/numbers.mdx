---
title: Numbers
description: Num Type available in KS
---

The `num` Type is KrunkScript's equivalent of JavaScript's [Number](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Data_structures#number_type).

## Calculations

```krnk
num value = 1 + 1;          # addition
num value = 10 - 1;         # subtraction
num value = 2 * 1.5;        # multiplication
num value = 2 ** 2;         # exponents
num value = 2 / 1;          # division
num value = 10 % 5;         # modulo / remainder
value += 10;                # add to value
value++;                    # increment value
value -= 5;                 # subtract from value
value--;                    # decrement value
num x = 3 << 1;             # bitshift left
num x = 3 >> 2;             # bitshift right
num x = 2 >>> 1;            # unsigned bitshift right
num value = 2 * (5 - 3);    # parentheses for order of operations
```
